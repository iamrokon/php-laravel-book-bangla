# Interview Questions: PHP Basics

**Q1: নিচের কোনটি পিএইচপির ডাটাটাইপ নয়?**
*Integer, Object, NULL, Number*
**Ans:** Number (পিএইচপিতে `Integer` এবং `Float` আছে, কিন্তু `Number` নামে আলাদা কোনো টাইপ নেই)।

**Q2: নিচের কোডটি চালালে আউটপুট কী হবে?**
```php
echo "10+10";
```
**Ans:** `10+10` (যেহেতু এটি স্ট্রিং)।

**Q3: What is the difference between `gettype` and `var_dump`?**
**Ans:** `gettype` শুধুমাত্র ডাটার টাইপ রিটার্ন করে, কিন্তু `var_dump` টাইপ এবং ভ্যালু উভয়ই রিটার্ন করে।

**Q4: PHP তে `null` এবং খালি স্ট্রিং `""` এর মধ্যে পার্থক্য কী?**
**Ans:** `null` হলো একটি স্পেশাল টাইপ যা বোঝায় ভেরিয়েবলটির কোনো মান নেই। `""` হলো একটি খালি স্ট্রিং যার মান আছে কিন্তু সেটি ফাঁকা।

**Q5: PHP তে অ্যারে তৈরির পদ্ধতি কী?**
**Ans:** `array()` ফাংশন দিয়ে অথবা `[]` শর্টকাট দিয়ে।

**Q6: অ্যারে কি `echo` দিয়ে প্রিন্ট করা যায়?**
**Ans:** না, `echo` শুধুমাত্র স্ট্রিং প্রিন্ট করতে পারে। অ্যারে দেখতে `print_r()` বা `var_dump()` ব্যবহার করা হয়।

**Q7: PHP তে ভিন্ন ভিন্ন টাইপের ডাটা একই অ্যারেতে রাখা যায়?**
**Ans:** হ্যাঁ, পিএইচপি অ্যারে হেটারোজিনিয়াস (Heterogeneous)।
