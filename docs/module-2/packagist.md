# Episode 6: Utilizing Packagist and Packages

**Packagist** হলো কম্পোজারের জন্য একটি কমিউনিটি-ড্রিভেন (community-driven) প্যাকেজ রিপোজিটরি (repository), যেখানে রয়েছে হাজার হাজার পিএইচপি প্যাকেজ।

`packagist.org` সাইট থেকে আমরা যে কোনো প্যাকেজ সার্চ দিয়ে বের করে আনতে পারব।

কমান্ড লাইন থেকে নিচের কমান্ডটি লিখে `Enter` প্রেস করলে আমরা available প্যাকেজগুলো দেখতে পাব:
```bash
composer search faker
```

এবার আমরা আমাদের প্রয়োজনীয় প্যাকেজগুলো ইন্সটল করতে পারি:
```bash
composer require fakerphp/faker
```

Packagist থেকে প্যাকেজ ইন্সটলের সময় কয়েকটা জিনিস মাথায় রাখতে হবে:
1. প্যাকেজটিকে অ্যাকটিভলি মেইনটেইন (actively maintain) করা হচ্ছে কিনা।
2. ভার্সনগুলো ঠিকমতো আপডেট হচ্ছে কিনা।
3. এতে আগে কোনো সমস্যা বা বাগ (bug) পাওয়া গিয়েছিল কিনা।
4. সিকিউরিটি (security) নিয়ে কোনো ঝুঁকি বা রিস্ক আছে কিনা।

packagist.org ছাড়াও `github`, `repman` ইত্যাদি রিপোজিটরি দিয়েও আমরা কম্পোজারের মাধ্যমে প্যাকেজ ইন্সটল করতে পারি। 

এমনকি আমরা চাইলে আমাদের নিজেদের তৈরি করা কোনো প্যাকেজও Packagist-এ সাবমিট (submit) করে পুরো কমিউনিটির জন্য উন্মুক্ত করতে পারি।
