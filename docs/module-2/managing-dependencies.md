# Episode 5: Managing Dependencies

কম্পোজার দিয়ে প্যাকেজ ম্যানেজ করার কিছু গুরুত্বপূর্ণ কমান্ড:

- `composer require vendor/package`: নতুন প্যাকেজ ইন্সটল করতে।
- `composer update`: প্যাকেজগুলো লেটেস্ট ভার্সনে আপডেট করতে।
- `composer remove vendor/package`: প্যাকেজ ডিলিট করতে।

ম্যানুয়ালি প্যাকেজ হ্যান্ডেল করার চেয়ে কম্পোজার ব্যবহার করা নিরাপদ এবং এরর হওয়ার সম্ভাবনা কম থাকে।
